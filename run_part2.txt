1. create and activate new virtural environment with Python 3.12 (conda, venv or other package manager)

2.  Download packages
	(GrantGuru/): pip install -r requirements.txt

Note that Phase 2 is designed to be ran as modules using Python modules from Phase2_work directory

3. Instantiate an empty DB:
	- Navigate to GrantGuru.Phase2_work
	- python3 -m src.system_functions.create_db_script

4. Running a test suite examplee:
	- python3 src.test_suites.users_rbp_test_suite

5. Deleting the DB instance
	- python3 -m src.system_functions.delete_db_script
